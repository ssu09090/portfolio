문제점

팀원 간의 Git 브랜치 관리 및 협업 방식의 이해도 차이로
병합 충돌과 데이터 관리 실수가 자주 발생했습니다.

해결방법

커뮤니케이션을 통해 풀/푸시 시간을 정해두고 병합을 시도했습니다.
또한 데이터를 효율적으로 관리하기 위한 방법을 서로 공유하며 진행했습니다.

개선방안

팀원 간 이해도를 맞추도록 Git 사용법, 브랜치 전략에 대해
공통 가이드 문서를 제작하고 공유할 계획입니다.
